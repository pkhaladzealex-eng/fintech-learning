# Day 90: Final Reflection and Portfolio Review

This document is an honest summary of my 90-day journey from zero coding knowledge to building a working fintech automation suite and active crowdsourced testing.

---

## 1. What I've Built
During these 90 days, I have successfully developed and packaged the following core deliverables:
* **Fintech Payment Testing Suite:** A multi-layered Python framework using Pytest and Selenium to automate web checkout flows.
* **API Validation Pipeline:** A backend integration script using the Stripe API to verify live transactions safely.
* **Database Ledger Logger:** A Python-SQLite handler that records payment statuses and protects against SQL injection.
* **Exploratory Bug Reports:** Real-world bug documentation submitted for top platforms like eToro, Kraken, and MoneyGram.
* **Support Framework:** A complete documentation set including `DEPLOYMENT_GUIDE.md`, `REFACTORING_NOTES.md`, and `FUTURE_ROADMAP.md`.

## 2. What I've Learned (Biggest Insights)
* **Real API and Libraries Mechanics:** I learned what an API actually is and how it functions. By using Python libraries, I can now assemble code blocks that talk to external servers, and seeing real, functional results is amazing.
* **Productive AI Collaboration:** I learned how to work with AI tools productively. Instead of wasting hours, I can now achieve much more in less time by asking the right technical questions during development.
* **The Big Picture of Code Architecture:** I realized that working code is not just about typing syntax. I learned to read code, analyze its structure, and see the big picture of how UI, APIs, and databases connect into one flow.

## 3. Skills Gained (Technical + Soft Skills)
* **Terminal & OS Ecosystem Integration:** Using the macOS Terminal on a daily basis is now stuck in my muscle memory. I fully understand how the local command line connects with Git and GitHub commands.
* **Backend Automation Routing:** Extracting Stripe API keys, connecting variables to the operating system, running scripts, committing, and uploading to GitHub has become my normal daily routine.
* **Cross-Platform Manual Testing:** Working hard on uTest since January 2026 taught me how to test on Windows, Android, macOS, and iPhone. I can now easily capture  charles proxy, console logs and browser logs from any device, and organize bugs using video/photo editors.
* **Professional QA Communication:** Through testing real applications like eToro and Kraken, I learned how to create clean bug reports, talk to TTLs (Test Team Leads), and handle strict deadlines, which was very stressful at first but taught me discipline.
